What is Dijkstra's Algorithm?

Dijkstra's algorithm is very similar to Prim's algorithm (for minimum spanning trees). It helps us find the shortest paths from a source to any vertice on a given weighted and directed graph. You can learn more about Djikstra's Algorithm here.

This is an example of a weighted and directed graph.

Steps
  1. Calculate distances (the cost) to adjacent nodes from source node.
  2. Pick next node with minimal distance; repeat adjacent node distance calculations.
  3. Finally, you will end up with the shortest-path tree from the source node to the node you want.
